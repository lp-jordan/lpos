# Workbook Rules

## Transcript Boundary
Each transcript represents one lesson within a course category.
Generate workbook content using ONLY the transcript for the current lesson. Do not reference or draw ideas from other transcripts.

## Content Accuracy
All takeaways and questions must be grounded in ideas that clearly appear in the transcript.
Do not introduce frameworks or advice that the speaker did not discuss.

## Audience
Write for thoughtful adult learners who value practical insight and real-world relevance.
Tone should be clear, grounded, professional, and reflective.

## Writing Rules
- Use only the transcript content
- Avoid generic motivational language
- Avoid the words: ensure, transparent, demonstrate, implement
- Do not use emojis
